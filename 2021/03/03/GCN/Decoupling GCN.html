Decoupling GCN with DropGraph Module for Skeleton-Based Action Recognition

发表于 2021-03-03 | 分类于 GNN | 阅读次数：
本文字数： 4.1k | 阅读时长 ≈ 4 分钟

Paper：https://www.ecva.net/papers/eccv_2020/papers_ECCV/papers/123690528.pdf
Github：https://github.com/kchengiva/DecoupleGCN-DropGraph

Main contributions

$\bullet$作者发现现有的GCN-based Skeleton-based action recognition method都受限于coupling aggregation mechanism（耦合聚合机制）。受CNNs中解耦聚合机制的启发，作者提出解耦图卷积网络**(DeCoupling Graph Convolutional Networks ,DC-GCN)**，在增加少数参数量（5%-10%）的前提下，boost the graph modeling ability。

$\bullet$另外一个GCN普遍的问题是过拟合，作者提出了DropGraph来丢弃相邻结点中的特征。此外，作者介绍了一种注意力指导丢弃机制（attention-guide drop mechanism,ADG）来增强正则化效果。

Approach

Decoupling Graph Convolutional Network (DC-GCN)

首先考虑单分区集的图卷积情况，实际上做了两个矩阵的乘法，$AX$和$XW$，其中，$AX$根据图的邻接矩阵计算不同骨架间的关系，作者称为spatial aggregation，而$XW$计算不同通道间的相关信息，作者称为channel correlation。在Fig.(a)中，spatial aggregation（$AX$）能被分解为分别计算每一个信道上的聚合。特征$X$的所有通道共享同一个邻接矩阵$A$（以相同颜色绘制）。所有目前的识别方法（ST-GCN，Nolocal adaptive GCN，AS-GCN，Directed-GCN）都采用这种方式，这种方式称为耦合图卷积（类似于退化深度卷积，它的卷积核在信道间被共享，退化深度卷积的表达能力明显弱于标准深度卷积）。

但是，CNN却不使用这种方式，而是使用解耦合卷积网络，这是CNN成功的关键，如图Fig.(b)所示，不同的信道有独立的空间聚合核。

于是，作者提出了解耦图卷积(DC-GCN)，不同的信道可以有独立且可训练的邻接矩阵，如图Fig.(c)所示。

但是与CNN的解耦合神经网络类似，网络中可能有冗余的卷积核，解耦合图卷积网络中可能会有冗余的邻接矩阵。因此作者把信道分离成$g$组（超参数）。在一组的信道共享一个可训练的邻接矩阵。当$g=C$时，每一个信道都有他自己能生成大量冗余参数的spatial aggregation kernel；当$g=1$时，DC-GCN退化成聚合图卷积(coupling graph convolution)。实验表明，超参数$g$的范围在8~16效果会比较好。在这种情况下，网络仅增加了5%~10%的参数。DC-GCN的公式如下：

Attention-guided DropGraph (ADG)

DropGraph的机制是当删除一个节点时，同样删除他的邻居节点集。其主要有两个参数$\gamma$用于控制样本概率；$K$控制删除邻居节点集的大小。

实现过程：

首先用概率为$\gamma$的伯努利分布对图中节点进行采样，然后删除根节点$V_{root}$以及和其最大距离为$K$的姐=节点。

作者用 $keep_prob$ 表示节点会被保留的概率，对于传统的dropout来说$keep_prob=1-\gamma$ ,但对于DropGraph来说，不仅有$V_{root}$被删除，相邻节点也有可能会被删除，所以 $keep_prob$应该与 $\gamma$和 $K$ 都有关。将每个节点的平均度定义为 $d_{ave}=2e/n$ ,随机抽样节点的第$i$阶邻域节点的期望个数可以估计为：$B_i=d_{ave} \times (d_{ave}-1)^{i-1}$

平均丢弃节点数为：$dropsize=1+\sum_{i=1}^KB_i$，则$\gamma= \frac{1-keep_prob}{dropsize}$（近似值，因为drop areas可能会有重叠。）

首先，保证$keep_prob$在（0.75-0.95），然后计算$\gamma$。

Spatial-temporal ADG

作者分别将ADG应用到空间图和时间图。ADG的输入时空特征为$\chi \in \mathbb{R}^{n \times C \times T}$。

空间图：作者通过在特征维度和时间维度上使用平均池化方法压缩$\chi$的绝对值，生成每个人体骨架$a_S \in \mathbb{R}^{n}$的spatial attention。接下来在$V_{root}$上采样，然后扩展到空间邻接点，再扩展到所有帧。

时间图：作者通过在特征维度和骨骼点维度上使用平均池化方法压缩$\chi$的绝对值，生成时间$a_T \in \mathbb{R}^{T}$的spatial attention.接下来在$V_{root}$上采样，然后扩展到时间邻接点，再扩展到全身所有节点。

注意：作者在GCN分支和跳连接分支使用ADG，是通过采样Dropblock: A regularization method for convolutional networks中的方法，将$keep_prob$逐渐从1降低到目标值。
